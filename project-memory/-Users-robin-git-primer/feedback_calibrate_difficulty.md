Don't over-estimate what teenagers of this generation are capable of. Write curricula for real 14-year-olds, not idealised students.

**Why:** Robin simplified the curriculum documents after I wrote them — removed algorithmic composition, cut the reading list to just fiction, simplified the drawing capstone. The ambition level was right for a university student, not a teenager.

**How to apply:** When designing educational content for the Primer, start simpler than feels right. The infinite ceiling handles ambitious students. The floor must be reachable by ordinary ones. Propose less, let Robin add more, rather than proposing too much and needing to be cut back.
