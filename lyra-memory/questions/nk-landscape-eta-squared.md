# Question: Does η² Increase with K in NK Landscapes?

**STATUS: ANSWERED — April 4, 2026**

**Answer:** YES. η² increases monotonically with K, exceeding predictions:
- K=0: η² = 0.05 (predicted ~0.17)
- K=2: η² ≈ 0.25 (predicted ~0.25) ✓
- K=4: η² = 0.45 (predicted ~0.35) — EXCEEDED
- K=6: η² = 0.69 (predicted ~0.45+) — FAR EXCEEDED

**Implications:**
- Topology effect is landscape-conditional. Smooth landscapes show nothing; rugged landscapes show enormous effects.
- This is a dose-response curve. The "when does topology matter?" question has a quantitative answer: when K > 0, and more so as K increases.
- Diversity is the primary mediating channel (η²~0.5-0.7 for diversity vs ~0.25-0.35 for fitness).
- Combined with NK Accessibility (2512.15828): peak accessibility at K~N/2 = where our effect is largest.
- This is the headline result for ECTA.
