## Web Research (arXiv + blogs)

### NEW Papers (not previously known)

**1. Graph-GRPO: Stabilizing Multi-Agent Topology Learning via Group Relative Policy Optimization**
- [arXiv:2603.02701](https://arxiv.org/abs/2603.02701) — Cang et al. (Tsinghua/Donghua), March 3, 2026.
- Group-relative policy optimization for learning communication topologies. Samples diverse graphs per query, computes edge advantage relative to group. Resolves credit assignment for individual communication edges.
- **Why it matters:** They need RL to discover what our categorical framework predicts structurally. Strengthens the argument for principled mathematical theory. **Potential ACT citation.**

**2. Conformity Dynamics in LLM Multi-Agent Systems: Topology and Self-Social Weighting**
- [arXiv:2601.05606](https://arxiv.org/abs/2601.05606) — Han et al., January 9, 2026.
- Hub topologies converge fast but are fragile (sensitive to hub quality). Distributed = robust but slower. "Wrong-but-sure cascades" from high connectivity.
- **Why it matters:** INDEPENDENT CONFIRMATION of strict/lax predictions! Centralized = strict (fast, fragile), distributed = lax (robust, slower). The self-social weighting parameter IS the strict-lax spectrum. **STRONG citation for both papers.**

**3. Categorical Belief Propagation: Sheaf-Theoretic Inference via Descent and Holonomy**
- [arXiv:2601.04456](https://arxiv.org/abs/2601.04456) — ter Horst, Mahadevan, Zambrano. January 8, 2026.
- CT foundation for belief propagation using free hypergraph categories, Grothendieck fibrations, sheaves. Exact inference = effective descent. HATCC algorithm detects descent obstructions via holonomy.
- **Why it matters:** Methodological parallel — using CT to explain when local computations compose into correct global results. Sridhar Mahadevan's name on it (UMass CT+AGI course). Sheaf obstruction = "when does island composition fail?"

**4. Operational Emergence of a Global Phase under Time-Dependent Coupling in Oscillator Networks**
- [arXiv:2603.05668](https://arxiv.org/abs/2603.05668) — Sanz. March 5, 2026.
- **DIRECTLY RELEVANT.** Algebraic connectivity (λ₂) governs synchronization onset. Faster coupling ramps cause "freeze-out." Different graph types COLLAPSE TO ONE CURVE when plotted against spectral parameter.
- **Why it matters:** This is EXACTLY our coupling onset result, arrived at independently from oscillator physics! λ₂ ordering matches our topology ordering. The spectral collapse = our cross-domain invariance. **HIGH-PRIORITY citation for both papers.**

**5. TopoCurate: Modeling Interaction Topology for Tool-Use Agent Training**
- [arXiv:2603.01714](https://arxiv.org/abs/2603.01714) — Yang et al. March 2, 2026.
- Projects multi-trial agent rollouts into "semantic quotient topology." Merges equivalent action-observation states. +4.2% SFT, +6.9% RL.
- **Why it matters:** The quotient topology construction is a categorical operation (coequalizer) done without knowing it. Evidence that practitioners need CT tools.

**6. Discovering MARL Algorithms with AlphaEvolve**

Zero overlap between CT audience and EC audience on Medium. The agent orchestration audience and the CT audience are separate readerships. Both the agent orchestration community and the CT community use "composability" — it's the lingua franca. But nobody defines it rigorously. Our opportunity: "What does composability actually mean mathematically in the agent context?"

The highest-quality content combines (a) real production failures with (b) a formal framework to understand them. Brinsa does (a) beautifully. Satyam does (b) aspirationally. Nobody does BOTH. Our article on composition failures through a CT lens would fill this exact gap.

### 5. Open-Endedness Has Gone Mainstream
