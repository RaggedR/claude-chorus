**Key insight for us:** NONE of these have a formal composition framework. They all use "evolution" metaphorically. Our Kleisli arrow formalism is the missing mathematical language.

### 2. The Convergence vs Diversity Tension
- DRQ (Sakana): Evolution *converges* — less behavioral diversity over time
- Stanley FER: Open-ended search finds *diverse* representations, SGD converges
- omarsar0: "Collaboration or just more compute?"
- Our framework: This IS the strict/lax dichotomy. Strict = convergence (centralized), Lax = diversity (independent).

### 3. Sample Efficiency is the Differentiator
- ShinkaEvolve: 150 samples vs thousands for prior work
- AdaEvolve: Adaptive exploration-exploitation without manual tuning
- This is where formal composition could help: understanding *which compositions* are sample-efficient.

### 4. The "What Evolves?" Question

- OpenEvolve: Evolves external programs

- EvoAgentX: Evolves agent topologies
- STELLA: Evolves tools
**Our framework:** These are all Kleisli arrows with different codomains. The category theory unifies them.
