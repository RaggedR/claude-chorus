# Connection #25: Emergence as Cohomology

> Created: 2026-03-17 (Dream Cycle)

## Source
Li et al. — "A Categorical Framework for Quantifying Emergent Effects in Network Topology" (arXiv:2311.17403, revised Jan 2025)

## The Connection
Li et al. formalize emergence as "structural nonlinearity" using homological algebra and derived functors. A system exhibits emergence when its global behavior cannot be reconstructed from local parts — measured by cohomological obstructions.

**Our laxator measures the same thing from the categorical side.** The laxator φ_G captures how much composition deviates from strict functoriality. Li's derived functors capture how much a global property deviates from being reconstructible from local data.

These are the SAME phenomenon expressed in different mathematical languages:
- **Laxator:** deviation from strict composition in a monoidal functor
- **Derived functors:** non-trivial higher cohomology (obstructions to global reconstruction)
- **Spectral gap:** algebraic connectivity governs synchronization onset (Sanz)
- **Fixation time:** graph structure determines consensus dynamics (Brewster/Nowak)

## Four-Way Mathematical Convergence
| Tradition | Framework | What it measures |
|-----------|-----------|-----------------|
| Category theory (us) | Kleisli/lax functors | Composition deviation |

| Physics (Sanz 2603.05668) | Coupled oscillators/λ₂ | Sync onset universality |

## Significance
When four mathematical communities using entirely different tools converge on the same structural result, that result is *real*. This is the strongest argument for the paper: we're not discovering an artifact of category theory. We're discovering a structural property of composed dynamical systems that any sufficiently expressive formalism will recover.
